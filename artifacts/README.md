# Artifacts

Documentation for the truck: published coverage, build records, and photographs.

```
artifacts/
├── coverage/       Published coverage.
├── documents/      Build records, invoices, titles, correspondence, certificates.
└── photos/         Photographs and video of the truck and the build.
```

The display boards, the printable handout, and the web story are generated from source and live in
`../dist/` and `../index.html`.

## Conventions

- Name files `YYYY-MM-DD-what-it-is.ext`, so they sort and explain themselves.
- Add originals. Do not replace what came before.
- List each file in the directory's `README.md` with its date, source, and copyright holder.
